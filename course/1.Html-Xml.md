#HTML

* HTML基础结构
* HTML标签    ``/<(\w+)>(.*?)(<\/\1>)?/``
    * 常用标签
    * 布局标签
    * 表单标签
* HTML属性    ``/<(\w+)\s*([\w-]+\="[^"]*"\s*)*>(.*?)(<\/\1>)?/i``
    * 4个通用属性
    * 表单标签属性
    * 特殊功能属性
    * 自定义属性
* HTML[实体](http://w3school.com.cn/tags/html_ref_entities.html)、[符号](http://w3school.com.cn/tags/html_ref_symbols.html)
* XHTML和HTML    ``/<(\w+)\s*([\w-]+\="[^"]*"\s*)*>(.*?)(<\/\1>)/``
    * no html, but xhtml
    * HTML验证: [http://validator.w3.org/](http://validator.w3.org/)
* HTML和XML      ``/<([:\w\-]+)>(.*?)(<\/\1>)/``
    * 从超文本的标记 到 可扩展标记语言
    * 可以做网页的 __数据传输格式__ ( ? )
